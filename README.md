## About TechGigs

TechGigs is a web application built with the expressive, elegant Laravel framework. TechGigs aims at connecting tech employers to their employees by creating a platform where employers can create/post gigs for developers to be able to search and apply.

## Basic Functionalities

1 Create Employer account/verification
2 Create/Modify gigs
3 Enabled search mode for the techGigs site
4 admin dashboard
